# MVP 0.1 — Поиск дешёвых товаров в маркетплейсах

## Цель
Пользователь вводит запрос -> получает список товаров из маркетплейсов -> может добавить в избранное -> перейти по ссылке на товар.

## MVP функции
- Регистрация/логин
- Поиск товаров по строке
- Список результатов: название, цена, маркетплейс, ссылка, картинка (если доступна)
- Избранное (добавить/удалить/просмотреть)
- История поисков (по пользователю)

## Не делаем в MVP
- Покупки внутри сервиса
- Уведомления о падении цены (подготовим модель данных, включим позже)
- Сложный “умный” matching одинаковых товаров между маркетплейсами
- Полноценный фронтенд — начнём с простого UI

## Нефункциональные требования
- Docker Compose окружение
- Postgres как основная БД
- Redis для кеша
- API документировано (Swagger/OpenAPI)
- Базовые тесты
